# Awesome Sales Pitches

A sibling repo for collecting reusable sales speech skills from famous pitches, standout marketing speeches, livestream selling, short-video hooks, and product use-case tests.

The repo is intentionally skill-first: each pattern should teach another agent how to turn a reference speech into a reusable, evidence-aware sales move.

## Structure

- `skills/three-angle-sales-pitch/` - turn one offer into proof-led, friction-led, and scene-led pitches.
- `skills/pitch-pattern-distiller/` - extract reusable pitch pattern cards from reference speeches.
- `test-runs/durian/` - verification fixture using selling durian instead of the original carrot example.
- `scripts/validate-skills.sh` - local validation for skill metadata and required fixtures.

## First Use Case

The first fixture uses fresh durian because it naturally exposes the three jobs a strong pitch must handle:

- Trust: "How do I know this is good?"
- Friction: "What about the smell, mess, and risk?"
- Scene: "Why would this be memorable enough to buy?"

## Rights Boundary

Use famous or celebrity speeches as sources for structure, not as copy to reuse. Do not invent endorsements, origin claims, health claims, or performance statistics.
