# Awesome Sales Pitches

A sibling repo for collecting reusable sales speech skills from famous pitches, standout marketing speeches, livestream selling, short-video hooks, and product use-case tests.

The repo is intentionally skill-first: each pattern should teach another agent how to turn a reference speech into a reusable, evidence-aware sales move. The default output contract is one selected pitch style, not a batch of every available angle.

## Structure

- `skills/pitch-style-selector/` - choose one neutral pitch style and apply it to a product.
- `skills/pitch-pattern-distiller/` - extract reusable pitch pattern cards from reference speeches.
- `test-runs/durian/` - verification fixture using selling durian instead of the original carrot example.
- `scripts/validate-skills.sh` - local validation for skill metadata and required fixtures.

## First Use Case

The first fixture uses fresh durian because it naturally verifies three selectable starter styles:

- `proof-ladder`: "How do I know this is good?"
- `honest-friction`: "What about the smell, mess, and risk?"
- `occasion-spark`: "Why would this be memorable enough to buy?"

## Rights Boundary

Use famous or celebrity speeches as sources for structure, not as copy to reuse. Style names should describe mechanisms, not people. Do not invent endorsements, origin claims, health claims, or performance statistics.
