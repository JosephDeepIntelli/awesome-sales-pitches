# Pattern Card Rubric

Use this rubric when reviewing distilled sales speech patterns.

## Required Fields

- Pattern name
- Source boundary
- Source move
- Reusable formula
- Best-fit products
- Failure modes
- Evidence needed
- Neutral example
- Applied use case
- Verification test

## Pass Criteria

- The pattern can be used on a different product without source wording.
- The pattern describes a persuasion mechanism, not merely a topic.
- The pattern exposes claims that need evidence before publishing.
- The pattern avoids fake celebrity endorsement or unverifiable authority transfer.

## Reject Criteria

- The "pattern" is only a summary of the source speech.
- The adapted example copies memorable phrasing from the source.
- The output depends on a celebrity name instead of the underlying speech move.
- The verification test only asks whether the copy "sounds good".
